# Contributing to LPLSP model
Thank you for your interest in contributing to the Lumped Parameter Linear Superposition (LPLSP) code.

## Project Goals
This repository aims to provide:
- Open-source implementations of the LPLSP methodology
- Sequential parameter estimation algorithms
- Ensemble parameter estimation algorithms
- Reproducible examples and benchmark studies

## Ways to Contribute
All forms of contributions are welcome.
Examples include:
- Bug reports
- Code improvements
- Documentation improvements
- Additional examples and tutorials
- Validation studies
- Benchmark datasets
- Performance optimization
- Porting algorithms to other languages

## Reporting Issues
Please open a GitHub Issue and provide:
- Description of the problem
- Script in Python / MATLAB
- Steps to reproduce the issue
- Expected behavior
- Actual behavior

## Coding Guidelines
- Use descriptive variable names.
- Add comments explaining important / non-obvious calculations.
- Document function inputs and outputs.
- Keep functions modular when possible.
- Maintain backward compatibility where practical.

## Documentation
When adding new functionality, please update:
- README.md
- Relevant files in docs
- Example scripts if applicable

## Research Contributions
If you are extending or applying the methodology in research, please consider:
- Referencing the relevant LPLSP publications.
- Documenting assumptions and limitations.
- Providing validation results where possible.
